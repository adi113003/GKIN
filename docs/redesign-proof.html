What this proof demonstrates, concretely: no gradient text, no background mesh, no scan-lines or blinking "live" dots; a single sober accent (ink-navy) with color used only for verdict meaning; an editorial serif for everything that's "voice," a clean grotesque for UI, and monospace reserved for genuine data (model IDs, dates, tiers). The analyzer reads top-to-bottom like a fact-check — verdict first, then the receipts, then optional depth — instead of eight side-by-side panels. Plain copy throughout ("Sign in," "Re-analyze," "History") replaces the "REGISTER AGENT / AUTHENTICATE / neural" register.
